Github A account to B account migration 
- git clone 
- check all the branch "git branch -a "
- add the remote branch for backup -> git remote add backup https://github.com/dinesh-careers/DhanGPT.git
- load all ur branch in new repo -> 

for branch in $(git branch -r | grep origin/ | grep -v HEAD | sed 's|origin/||'); do
    git push backup "refs/remotes/origin/$branch:refs/heads/$branch"
done
